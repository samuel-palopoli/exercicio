// 3. Saída de Dados:
//    - Apresente o maior e o menor valor encontrados.

// Exemplos:

// - Exemplo 1:
//   - Entrada: 10, 5, 8, 12, 3

//   - Saída: Maior valor: -1, Menor valor: -8

// - Exemplo 3:
//   - Entrada: 100, 200, 150, 300, 250
//   - Saída: Maior valor: 300, Menor valor: 100

//     - menor_valor = num2
