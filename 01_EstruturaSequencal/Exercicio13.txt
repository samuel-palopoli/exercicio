// Exercício 13: Cálculo da Soma dos Quadrados de Três Números

// Descrição:
// Este exercício consiste em desenvolver um programa que leia três valores
// numéricos, calcule a soma dos quadrados desses números e exiba o resultado.
// O programa receberá como entrada três números, calculará o quadrado de cada
// um deles, somará os quadrados e, por fim, exibirá o resultado da soma.

// Passo a Passo:

// 1. Entrada de Dados:
//    - Solicite ao usuário que insira três valores numéricos.
//    - Aguarde a entrada do usuário.

// 2. Processamento:
//    - Calcule o quadrado de cada um dos três números.
//    - Some os quadrados calculados.

// 3. Saída de Dados:
//    - Exiba o resultado da soma dos quadrados.

// Exemplos:

// - Exemplo 1:
//   - ENTRADA: 2, 3, 4
//   - SAÍDA: 29 (2² + 3² + 4² = 4 + 9 + 16 = 29)

// - Exemplo 2:
//   - ENTRADA: 1, 5, 6
//   - SAÍDA: 62 (1² + 5² + 6² = 1 + 25 + 36 = 62)

// - Exemplo 3:
//   - ENTRADA: -3, 0, 2
//   - SAÍDA: 13 ((-3)² + 0² + 2² = 9 + 0 + 4 = 13)

// Exemplo em Pseudocódigo:
// ENTRADA: num1, num2, num3
// PROCESSAMENTO: soma_quadrados = (num1 * num1) + (num2 * num2) + (num3 * num3)
// SAÍDA: exibir soma_quadrados
