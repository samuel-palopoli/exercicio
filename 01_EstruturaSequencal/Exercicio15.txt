// Exercício 15: Cálculo do Produto e da Soma de Quatro Números

// Descrição:
// Este exercício consiste em desenvolver um programa que leia quatro valores
// numéricos, calcule o produto do primeiro com o terceiro e a soma do segundo
// com o quarto, e exiba os resultados do produto e da soma. O programa receberá
// como entrada quatro números, realizará os cálculos especificados e exibirá os
// resultados do produto e da soma.

// Passo a Passo:

// 1. Entrada de Dados:
//    - Solicite ao usuário que insira quatro valores numéricos.
//    - Aguarde a entrada do usuário.

// 2. Processamento:
//    - Calcule o produto do primeiro e do terceiro número.
//    - Calcule a soma do segundo e do quarto número.

// 3. Saída de Dados:
//    - Exiba os resultados do produto e da soma.

// Exemplos:

// - Exemplo 1:
//   - ENTRADA: 2, 3, 4, 5
//   - SAÍDA: Produto: 8 (2 * 4), Soma: 8 (3 + 5)

// - Exemplo 2:
//   - ENTRADA: -1, 8, 0, 2
//   - SAÍDA: Produto: 0 ((-1) * 0), Soma: 10 (8 + 2)

// - Exemplo 3:
//   - ENTRADA: 10, 5, 3, 7
//   - SAÍDA: Produto: 30 (10 * 3), Soma: 12 (5 + 7)

// Exemplo em Pseudocódigo:
// ENTRADA: num1, num2, num3, num4
// PROCESSAMENTO: produto = num1 * num3
//                soma = num2 + num4
// SAÍDA: exibir "Produto: " + produto + ", Soma: " + soma
